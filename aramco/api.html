Meeting API and ACEA requirements
Interchangeable and linked group II base oils.
In accordance with API and ACEA protocols, Saudi Aramco has successfully completed test programs that demonstrated the interchangeability and linkage between Group II base oils from our global manufacturing sites. The extensive testing programs covered API SN/GF-5, API CJ-4 and ACEA A3/B4 automotive engine oils together with demonstration in industrial and gear oils applications. Engine tests to meet ACEA E7 and E9 are currently underway with completion expected Q1 2019.
